# Oracle R2DBC code examples
A simple R2DBC program that connects to ADB and executes a query

# Building manually

|        | Task  | Command                                          |
| ------ | ----- | ------------------------------------------------ |
| Maven  | Build | `mvn clean install`                              |
|        | Run   | `mvn exec:java`                                  |
| Gradle | Build | `gradle build`                                   |
|        | Run   | `gradle run`                                     |
